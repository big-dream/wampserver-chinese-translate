## 使用方法
下载仓库所有文件，解压到wampserver安装目录下。然后鼠标右键单击电脑通知栏的`wampmanger`图标-`语言(Language)`-`chinese`切换到简体中文。

## 其它
* 所有文件均基于原Wampserver程序内的原版文件进行翻译
* 转载请注明出处（翻译和维护不易，注明出处仅仅一两行文字，不会很占用你多少版面）
